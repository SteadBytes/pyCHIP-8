# pyCHIP-8
COSMAC ELF CHIP-8 interpreter emulator for Python3

## Usage
Pass binary ROM to run as first argument in terminal:
 `$ python main.py "path_to_ROM"`
## Key Mapping
```
CHIP-8      --->    pyCHIP-8
-------             --------
1 2 3 C     --->    1 2 3 4
4 5 6 D     --->    Q W E R
7 8 9 E     --->    A S D F
A 0 B F     --->    Z X C V
```
